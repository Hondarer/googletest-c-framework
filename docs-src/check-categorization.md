# 確認の細分化

## 概要

テスト結果における確認件数のカウントを、正常系, 準正常系, 異常系, カテゴリ未指定 の 4 つのカテゴリに分類してカウントし、分類別に集計結果を表示する機能です。

## 利用可能なタグ

### カテゴリ未指定

- `[確認]`
- `[Pre-Assert確認]`

### 正常系

- `[確認_正常系]`
- `[Pre-Assert確認_正常系]`

### 準正常系

- `[確認_準正常系]`
- `[Pre-Assert確認_準正常系]`

### 異常系

- `[確認_異常系]`
- `[Pre-Assert確認_異常系]`

## 計数方法

確認件数のカウントを、正常系, 準正常系, 異常系, カテゴリ未指定 の 4 つに分類して計数します。

## 結果出力

### カテゴリ未指定のみの場合

```text
### 確認内容 (n)
### 確認内容 (n * m)  # Theory テスト時（.NET のみ）
```

### カテゴリ分類が存在する場合

カテゴリタグ（`_正常系`、`_準正常系`、`_異常系`）を 1 つでも使用している場合、分類別の表示になります。

```text
### 確認内容 (正常系:n1, 準正常系:n2, 異常系:n3, カテゴリ未指定:n4)
### 確認内容 (正常系:n1 * m, 準正常系:n2 * m, 異常系:n3 * m, カテゴリ未指定:n4 * m)  # Theory テスト時
```

0 件のカテゴリは表示されません。

**例**: 正常系と異常系のみの場合

```text
### 確認内容 (正常系:2, 異常系:1)
```
